Jeison Bainha de Oliveira

Java | Full Stack Developer


Portuguese-Brazilian IT professional based in Portugal, currently transitioning into software development after nearly a decade working as a lawyer leading the regulatory area of the oil & gas sector.

My analytical background in exact sciences—developed during my time in the Brazilian Navy, through Mechanical Engineering studies, and years teaching physics and calculus—shaped a structured and objective approach to problem solving that I now apply to software engineering.

I am currently attending the Software Developer program at CESAE-Digital, a training path oriented toward the job market with a strong focus on Java programming and Object-Oriented Programming (OOP), as well as modern Web Development. The program also covers technologies such as JavaScript, React, Next.js, and mobile development concepts using Swift and Kotlin.

Alongside the course, I have been actively building full-stack, cloud-ready web applications, working with React, Next.js, and Neon (serverless database).

Previously, I deepened my technical expertise through specialized training at the University Institute of Lisbon – ISCTE, where I obtained a Java programming certification. During that period, I developed a Personal Finance Management System using Java and Object-Oriented Programming in IntelliJ, built a robust information system using MySQL, and authored a technical report accompanied by a Spring Boot application, coordinated through Jira and designed using Figma.

My most recent professional experience includes supporting data migration projects through the Anypoint Platform (MuleSoft) and collaborating within Agile teams using Azure DevOps.

I also hold certifications from Microsoft (Azure Fundamentals) and Google (Cloud Digital Leader), reinforcing my commitment to staying at the forefront of cloud technologies and modern software development.
